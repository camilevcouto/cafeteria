Coffee Bronw 
Este projeto foi elaborado para a faculdade, tendo como propósito criar um sistema completo com login, apresentação de produtos, carrinho de compras e etc.
Tecnologias utilizadas:
HTML, CSS, JAVASCRIPT E PHP.

Documentação do projeto: 
O problema inicial girou em torno do desapontamento com uma empresa no desenvol-vimento de software de um e-commerce de uma cafeteria agora propus a criação de um sistema e-commerce de uma cafeteria em linguagem JavaScript e PHP com uma estrutura HTML e apresentação em CSS e um banco de dados MySQL com funciona-lidades e gerenciamento de produtos visando uma melhora no sistema.
